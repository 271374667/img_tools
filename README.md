一个简单的命令行图片处理工具，提供命令行交互版本以及纯命令行命令版本，适用于对图片进行批量处理，用于我日常逛Pixiv的时候对下载来图像进行排序和处理，支持下面的这些功能

- 图片压缩
- 图片统一旋转(例如统一竖屏或者统一横屏)
- 图片格式转换
- 图片批量去重
- 图片超分

以上的这些功能基本均支持批量处理，可以传入文件夹也可以传入单个图片的路径
